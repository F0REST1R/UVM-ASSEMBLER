# Язык ассемблера Учебной Виртуальной Машины (УВМ)

## Синтаксис команд

### 1. Загрузка константы

LOAD <регистр>, <константа>

**Пример:**
```asm
LOAD R9 771    ; Загрузить константу 771 в регистр 9
```

### 2. Чтение из памяти

READ <регистр_результата>, <смещение>, <базовый_регистр>

**Пример:**
```asm
READ R35 499 R42  ; Прочитать значение из адреса [R42 + 499] в R35
```

### 3. Запись в память

WRITE <регистр_значения>, <регистр_адреса>

**Пример:**
```asm
WRITE R25 R3       ; Записать значение из R25 по адресу из R3
```

### 4. Квадратный корень

SQRT <регистр_источника> <адрес_результата>

**Пример:**
```asm
SQRT R9 804        ; Вычислить sqrt(R9) и записать по адресу 804
```
